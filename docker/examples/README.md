# Examples

Docker Compose examples for running Autoware with the `docker-new` images.

## Basic

Minimal dev shells — `dev-cpu`, `dev-dri`, `dev-nvidia`. Pick the one matching your GPU. See [basic/README.md](basic/README.md).
